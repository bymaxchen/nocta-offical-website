# Nocta Official Website

A modern, animated company website built with Next.js and Tailwind CSS.

## Features
- Animated, auto-scrolling company logo bar
- Responsive header with hide-on-scroll and animated menu button
- Fullscreen menu with expandable/collapsible sections
- Clean, modern UI

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Visit [http://localhost:3000](http://localhost:3000)

## Tech Stack
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Project Structure
- `src/app/components/` — Reusable UI components
- `public/logos/` — Company logo images

## Customization
- Edit `src/app/components/LogoScroller.tsx` to change the logos
- Edit `src/app/components/FullScreenMenu.tsx` to change menu items
